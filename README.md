# Kioku Anki Add-on

This Anki add-on works with the Kioku Chrome extension to automatically add mined Japanese sentences as flashcards.

## Installation

1. **Open Anki** (latest version recommended).
2. Go to **Tools > Add-ons** (or press `Ctrl+Shift+A`).
3. Click **View Files**.
4. Download this add-on’s source code and **paste it** into the folder that opens.
5. Restart Anki to activate the add-on.
